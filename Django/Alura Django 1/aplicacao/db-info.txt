PostgreSQL DB info
  Server = dbserver
  Database = alura_receita
Import the database from another machine (on Linux, without a PgAdmin GUI)
	sudo passwd postgres
	su postgres
	pg_dump -U postgres -W -Fc db > db.dump
	pg_restore -U postgres -W -d db db.dump
	(this will also restore the django admin user/password)
Make the django migrations
	python manage.py runserver
	python manage.py makemigrations
	python manage.py migrate
