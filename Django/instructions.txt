Install
	Python3
	PostgreSQL
Create the Pythyon Virtual Environment
	python -m pip install --upgrade pip
	python -m venv ./venv
Start the Venv
	* CMD
	  venv\Scripts\activate.bat
	* Powershell
	  Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
	  venv\Scripts\Activate.ps1
	* Linix/Mac
	  source ./venv/bin/activate
Install the requirements in the venv
	python -m pip install --upgrade pip
	pip install -r pip_requirements.txt
Configure the PostgreSQL database
	Create new server (dbserver)
	Create a new database (alura_receita)
On Linux you might need to change the postgres user password
	sudo -u postgres psql
	ALTER USER postgres PASSWORD 'password_here';
	\q
Import the database from another machine (on Linux, without a PgAdmin GUI)
	sudo passwd postgres
	su postgres
	pg_dump -U postgres -W -Fc db > db.dump
	pg_restore -U postgres -W -d db db.dump
	(this will also restore the django admin user/password)
Make the django migrations
	python manage.py runserver
	python manage.py makemigrations
	python manage.py migrate
Create a django superuser if needed
	python manage.py createsuperuser
Access the admin panel from the browser
	localhost:8000/admin
